# Market Scout - Industry Scoring Tool

A React-based tool to evaluate industry viability based on adoption friction and price sensitivity.

## Features

- Interactive assessment with 10 key questions
- Visual scatter plot analysis
- Quadrant-based market positioning
- Mobile-responsive design
- Clean, modern UI with Tailwind CSS

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Recharts (data visualization)
- Lucide React (icons)

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

This project is configured for easy deployment on Vercel:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Vercel will auto-detect the Vite configuration
4. Deploy!
